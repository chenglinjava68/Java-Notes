#### 间隙锁

间隙锁（gap lock）是为了解决幻读引入的。**它锁的是两个值之间的空隙。间隙锁之间不存在冲突关系。跟间隙锁存在冲突 关系的，是“往在这个间隙中插入一个记录”这个操作**。

需要注意的是，间隙锁在**可重复读隔离级别下才有效**，并且使用间隙锁会出现**死锁**的情况！！！
